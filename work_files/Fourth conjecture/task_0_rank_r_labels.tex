\section*{Task 0: Rank-`r` labels}

\paragraph{Goal.}
Identify the subset of lifted cyclic rank patterns relevant to the rank-\(r\)
critical locus attached to the singular subset \(S\).

\paragraph{Admissible labels for the fixed-arrow slice.}
From the third-conjecture block, the fixed-arrow cyclic locus
\[
\Gamma_{d,S}
\]
is classified by the lifted cyclic rank patterns satisfying
\[
\rho_{L,L+1}=d_L-r.
\]
This is the closing-arrow constraint forced by
\[
W_0=\Sigma_{XY}P_Q.
\]

\paragraph{Adding the end-to-end rank.}
Inside the rank-\(r\) cyclic locus
\[
\Gamma_{d,S}^r,
\]
one must also impose
\[
\rho_{0L}=r.
\]
Therefore the relevant label set is
\[
\widetilde{\mathcal R}_{d,S}^{\,r}
:=
\{\rho\in \widetilde{\mathcal R}_d^{\mathrm{wt}}
\mid
\rho_{L,L+1}=d_L-r,\ \rho_{0L}=r\}.
\]

\paragraph{Remark on notation.}
Here the same scalar \(r\) appears in two ways:
\begin{enumerate}
\item the closing-arrow rank equals \(d_L-r\) because \(|S|=r\);
\item the open-chain rank equals \(r\) because we are restricting to
  \(\Gamma_{d,S}^r\).
\end{enumerate}
This is exactly the DLN regime relevant for the critical locus
\(\mathcal C_{d,S}^r\).

\paragraph{Status.}
Completed. The main theorem will be indexed by the finite set
\(\widetilde{\mathcal R}_{d,S}^{\,r}\).
